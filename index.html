3D mobile puzzle game prototype

Unity, C#

Goal of this game is to push boxes in correct order and directions so it’s possible for player to reach the end of the level. Player can only push one box at the same time. Both player and boxes are affected by gravity. Movement happens gradually between invisible grid points.

Platformer game prototype

Slick2D, Java

Goal of this game is to reach end of the level while avoiding placed obstacles. Player is affected by gravity, can move left and right and also jump. Levels of this game are built using a tile map. There is a level editor, which allows to easily build levels and save them in text files.

"Worms" type game prototype

Slick2D, Java

Turn-based game which has functionality for player to move around the level, jump, shoot rockets and deal damage to others. Game level is loaded from a picture and mask and can be completely destroyed using the given weapon. Both player and rockets are affected by gravity. Movement is programmed using pixel detection.
